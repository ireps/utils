Command	Result:

CTRL + a + -	        Split the active pane horizontally
CTRL + a + |	        Split the active pane vertically
CTRL + a + arrow key	Switch to another pane
CTRL + a + ALT+arrow	Resize the active pane
CTRL + a + z	        Zoom in on the active pane. Press the same combination again to exit zoom mode
exit	                Close the active pane
CTRL + a + x	        Force kill an unresponsive process in a pane
